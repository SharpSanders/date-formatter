# Date Formatter

A small JavaScript project that displays today's date in different formats.  
Built as part of a learning path to practice the JavaScript `Date` object, DOM manipulation, and basic styling.

## Demo

The app shows today's date and lets you switch between different display formats using a dropdown:

- `Day-Month-Year` (default)
- `Year-Month-Day`
- `Month-Day-Year Hours Minutes`

## Features

- Uses the built-in [`Date`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date) object
- Dynamically updates the date text when the select input changes
- Simple, responsive layout with a custom SVG calendar icon
- Clean, accessible UI using plain HTML, CSS, and JavaScript

## Tech Stack

- **HTML5** – structure and content (`index.html`)
- **CSS3** – layout and styling (`styles.css`)
- **Vanilla JavaScript** – logic and DOM interaction (`script.js`)
